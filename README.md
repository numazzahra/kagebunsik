# Kagebunsik

Kagebunsik is a responsive food ordering web application developed to sell Korean food around UPN Veteran Jakarta. The platform streamlines the ordering process with a modern and user-friendly interface.

## Features

- Browse food menus
- View product details
- Shopping cart
- Checkout process
- Responsive design

## Tech Stack

- Vue.js
- Vite
- JavaScript
- CSS

## Installation

```bash
npm install
npm run dev
```
